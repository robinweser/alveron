# Alveron TodoList Example

<img src="https://github.com/rofrischmann/alveron/blob/master/docs/res/TodoApp.gif">

## Installation

To install the dependencies, run:

```sh
# yarn
yarn

# npm
npm i
```

## Running

To start the Next.js server providing the example you may run:

```sh
# yarn
yarn start

# npm
npm start
```
