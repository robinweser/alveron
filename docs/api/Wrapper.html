5.4. Wrapper

Wrapper

Wrapper is just a convenient component that combines both Provider and Consumer into a single component.
It helps to keep your component tree clean when using local state.

It accepts all props that Provider and Consumer accept and requires a render function passed as children.

Example

import { createStore } from 'alveron'

const model = 0
const actions = {
  increment: state => state + 1,
  decrement: state => state - 1
}

const Counter = createStore({
  actions,
  model
})

const renderCounter = ({ state, actions }) => (
  <div>
    <div>Counter: {state}
    <button onClick={actions.increment}>+</button>
    <button onClick={actions.decrement}>-</button>
  </div>
)

const SimpleCounter = ({ defaultCount }) => (
  <Counter.Wrapper>{renderCounter}</Counter.Wrapper>
)
