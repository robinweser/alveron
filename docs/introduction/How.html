1.2. How It Works

How It Works

Woodworm is entirely built on React's build-in [Context API].
Creating a new store instance using createStore will use React's createContext to create a local context store returning two components: Provider and Consumer.
Woodworm wraps those components in order to automatically bind the model, actions and effects to its local React state.
